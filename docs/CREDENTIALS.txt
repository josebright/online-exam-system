================================================================================
TESTING WORKFLOW
================================================================================

AS ADMIN:
1. Login with admin credentials
2. Navigate to "Manage Exams"
3. Create a new exam or edit existing one
4. Add questions (try all 5 types)
5. Publish the exam
6. View analytics dashboard

AS STUDENT:
1. Login with student credentials (or register new account)
2. View available exams on dashboard
3. Click "Start Exam" on any published exam
4. Read instructions and start
5. Answer questions (timer is running)
6. Submit exam or let it auto-submit
7. View detailed results
