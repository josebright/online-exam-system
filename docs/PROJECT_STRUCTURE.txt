Timer Synchronization:
  Client Timer → Poll Server (every 30s) → Server calculates remaining time
  → Return to client → Update display → Auto-submit if expired

Auto-save Flow:
  User answers → Collect answers → AJAX POST (every 30s) → Validate CSRF
  → Store in database → Return success → Update UI indicator

Tab Switch Detection:
  Visibility change event → AJAX POST → Increment counter → Store in DB
  → Show warning if threshold exceeded → Admin can view count

┌─────────────────────────────────────────────────────────────────────────┐
│                         SECURITY FLOW                                    │
└─────────────────────────────────────────────────────────────────────────┘

Request Flow:
  User Input → Client Validation → Server Validation → CSRF Check
  → XSS Sanitization → SQL Injection Prevention → Business Logic
  → Database (Prepared Statements) → Response → Output Sanitization
  → Display to User

Session Security:
  Login → Generate secure token → HTTP-only cookie → Session expiration
  → IP tracking → Activity logging → Auto-logout on timeout
